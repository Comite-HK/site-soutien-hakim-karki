# Comité de soutien au juge Hakim Karki

Site officiel du comité de soutien au juge Hakim Karki.

## Objectif du site

Informer, documenter et permettre au public de comprendre les enjeux de l’affaire Hakim Karki.

Le site a pour vocation de centraliser :

* les informations publiques,
* la chronologie de l’affaire,
* les éléments de contexte,
* les prises de position,
* la revue de presse,
* les actions de soutien.

---

## Technologies utilisées

* HTML5
* CSS3
* JavaScript
* GitHub Pages

---

## Structure du site

* Accueil
* Le Comité de soutien
* Chronologie
* Revue de presse
* Écrire à Hakim
* Agir / partager
* Documents et ressources

---

## Direction du projet

Le site est conçu comme une plateforme documentaire et citoyenne moderne.

Objectifs principaux :

* améliorer la compréhension publique de l’affaire,
* offrir une navigation claire et accessible,
* garantir une consultation fluide sur mobile et ordinateur,
* assurer une présentation sobre, crédible et professionnelle.
